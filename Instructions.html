🛠 Príprava na prednášku o Raymarching

Ahojte! Na ďalšom cvičení budeme písať vlastný 3D render od nuly, pričom využijeme len matematiku a grafickú kartu. Aby sme stihli vytvoriť peknú animovanú „Lávovú lampu“, pripravte si prosím pracovné prostredie ešte pred začiatkom hodiny.

Zaberie to približne 10-15 minút.

Krok 1. Inštalácia základného softvéru

Ak už máte nainštalovanú Javu a vývojové prostredie, tento krok môžete preskočiť.

  1. Uistite sa, že máte nainštalovaný Java Development Kit (JDK) verzie 17 alebo novšej.
  2. Stiahnite si a nainštalujte IntelliJ IDEA (postačí bezplatná Community Edition).

Krok 2. Príprava knižnice LWJGL (OpenGL)

Pre prácu s grafikou budeme používať knižnicu LWJGL.

  1. Stiahnite si archív s knižnicami z tohto odkazu: [Odkaz na stiahnutie archívu]
  2. Rozbaľte priečinok na vhodné miesto vo vašom počítači (napríklad C:\LWJGL).
    Dôležité: cesta k priečinku nesmie obsahovať diakritiku (mäkčene, dĺžne) ani cyriliku!

Krok 3. Vytvorenie projektu v IntelliJ IDEA

  1. Otvorte IntelliJ IDEA a kliknite na New Project.
  2. V ľavom menu vyberte New Project (obyčajný Java projekt, bez Maven/Gradle).
  3. Zadajte názov projektu, napríklad RaymarchingLavaLamp.
  4. Uistite sa, že v poli Language je vybraná Java, a v poli Build system — IntelliJ.
  5. Kliknite na Create.

Krok 4. Pripojenie knižnice k projektu

Nepotrebujeme všetky moduly zo stiahnutého priečinka, stačia nám len tri základné:

  1. V hornom menu IDEA vyberte File -> Project Structure (alebo stlačte Ctrl + Alt + Shift + S).
  2. V ľavom stĺpci vyberte sekciu Modules, potom prejdite na kartu Dependencies.
  3. Kliknite na ikonu + (plus) a vyberte JARs or directories...
  4. V otvorenom okne nájdite váš rozbalený priečinok lwjgl-3.3.6. Vojdite doň a označte súbory .jar LEN z týchto troch priečinkov (označte aj obyčajné súbory, aj tie s príponou natives):
    • 📁 lwjgl
    • 📁 lwjgl-glfw
    • 📁 lwjgl-opengl
  5. Kliknite na OK, potom Apply a znova OK.

Krok 5. Testovací kód

Poďme sa uistiť, že všetko funguje správne.

  1. V priečinku src vášho projektu vytvorte novú Java triedu s názvom PracticeTask.
  2. Skopírujte a vložte do nej nasledujúci kód:
import org.lwjgl.glfw.GLFW;
import org.lwjgl.glfw.GLFWErrorCallback;

public class PracticeTask {
    public static void main(String[] args) {
        // Nastavenie výpisu chýb do konzoly
        GLFWErrorCallback.createPrint(System.err).set();

        // Pokus o inicializáciu GLFW
        if (!GLFW.glfwInit()) {
            throw new IllegalStateException("Chyba: Nepodarilo sa inicializovať GLFW. Skontrolujte pripojenie knižníc!");
        }

        System.out.println("Hurá! Knižnica LWJGL je úspešne pripojená a funguje.");
        
        // Ukončenie práce
        GLFW.glfwTerminate();
    }
}

🚨 Krok 6. DÔLEŽITÉ pre majiteľov Mac (macOS)

Používatelia systémov Windows a Linux môžu tento krok preskočiť.

Architektúra macOS vyžaduje, aby grafické okná boli vytvorené v hlavnom vlákne (main thread). Ak kód spustíte len tak, dostanete chybu GLFW may only be used on the main thread.

  1. V pravom hornom rohu IDEA kliknite na rozbaľovací zoznam PracticeTask a vyberte Edit Configurations...
  2. Nájdite pole VM options. Ak tam nie je: kliknite na Modify options (alebo na ikonu ozubeného kolesa) -> vyberte Add VM options.
  3. Napíšte tam presne jeden riadok:
    -XstartOnFirstThread
  4. Kliknite na Apply a potom OK.

Krok 7. Spustenie!

Spustite kód (zelený trojuholník vedľa metódy main).

Ak sa v konzole (v spodnej časti obrazovky) zobrazí správa «Hurá! Knižnica LWJGL je úspešne pripojená a funguje.» — ste plne pripravení na hodinu!
